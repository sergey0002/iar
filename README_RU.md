# IAR — Image Auto Resizer

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PHP Version](https://img.shields.io/badge/php-%3E%3D7.4-8892bf.svg)](https://php.net)

**iar** — это самодостаточный PHP-движок для динамического изменения размеров, кадрирования и автоматической конвертации форматов (WebP/AVIF). Создан для разработчиков, которым нужно мощное, быстрое и переносимое решение без лишних зависимостей.

## Пример использования

/products/hero.jpg - оригинальный путь к изображению.

<img src="/iar/products/hero.jpg?w=1200" /> Если изображение больше 1200 в ширину оно будет уменьшено с сохранением пропорций до ширины 1200 px

Ресайз делается один раз, миниатюра сохраняется в кеш, и при повторном вызове обращение будет передано напрямую к файлу кеша, при изменении оригинального файла файл кеша будет обновлен автоматически.

<img src="/iar/products/hero.jpg?w=1200"
     srcset="/iar/products/hero.jpg?w=480 480w,
             /iar/products/hero.jpg?w=800 800w"
     sizes="(max-width: 600px) 480px, 800px"
     alt="SEO Оптимизированное фото">

## 🌟 Почему iar?

- **⚡ Скорость**: Прозрачное кеширование и поддержка ETag. Повторные запросы отдаются за миллисекунды.
- **🛡️ Безопасность**: Защита от Path Traversal и DoS. Белые списки параметров и строгая проверка путей.
- **📦 Ноль системных зависимостей**: Нужен только PHP + GD. Не требует глобально установленного `composer` или `imagemagick`.
- **🚀 Современные стандарты**: Авто-выбор WebP/AVIF на основе поддержки браузером (`Accept` header).
- **📁 Поддержка подпапок**: Работает с любой глубиной вложенности ваших изображений.

## 🚀 Быстрый старт

### 1. Установка

`iar` требует библиотеку **Intervention Image v3**. Вы ДОЛЖНЫ установить зависимости через Composer внутри папки проекта.

#### Способ А: Через Git

```bash
git clone https://github.com/sergey0002/iar.git
cd iar
composer install --no-dev --optimize-autoloader
```

#### Способ Б: Вручную (Альтернативный)

1. **Скопируйте папку** `iar` в корень вашего сайта.
2. **Откройте терминал** и перейдите **ВНУТРЬ** папки `iar`:

   ```bash
   cd путь/к/вашему/сайту/iar
   ```

3. **Установите зависимости**:
   *Если у вас есть глобальный `composer`:*

   ```bash
   composer install --no-dev --optimize-autoloader
   ```

   *Если у вас **НЕТ** глобального composer:*

   ```bash
   # Скачиваем локальный composer.phar и устанавливаем
   php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"
   php composer-setup.php
   php -r "unlink('composer-setup.php');"
   php composer.phar install --no-dev --optimize-autoloader
   ```

> [!NOTE]
> Папка `cache/` будет создана автоматически при первом запросе. Убедитесь, что у скрипта есть права на запись в своей директории.

Папку /iar/ в корне сайта к которой будут идти запросы вы можете переименовать по своему усмотрению все будет работать корректно.

### 2. Базовое использование (SEO-ссылки)

Обращайтесь к изображениям напрямую по их пути. Система сама поймет, что нужно сделать ресайз.

- **Ресайз**: `/iar/photo.jpg?w=800`
- **Работа с подпапками**: `/iar/products/2026/iphone15.jpg?w=400` где /products/2026/iphone15.jpg оригинальный путь к изображению
- **Любая вложенность**: `/iar/assets/img/ui/icons/logo.png?w=100`

### 3. Все возможности системы

| Возможность | Описание | Пример |
| :--- | :--- | :--- |
| **Ресайз** | Пропорциональное изменение по ширине или высоте | `?w=800` или `?h=600` |
| **Умный кроп** | Режимы `fit`, `cover` (с гравитацией), `resize` | `?w=400&h=400&m=cover` |
| **Гравитация** | Привязка кропа: `top`, `bottom`, `left`, `right` | `?m=cover&g=top` |
| **Обрезка краёв** | Обрезка пикселей/процентов (`ct`, `cr`, `cb`, `cl`) | `?ct=10p` (10% сверху) |
| **Свой формат** | Принудительная конвертация в нужный тип | `?fm=webp` или `?fm=avif` |
| **Авто-формат** | Выдача WebP/AVIF современным браузерам | *Включено по умолчанию* |

### 4. Пример для верстки (`srcset`)

```html
<img src="/iar/products/hero.jpg?w=1200" 
     srcset="/iar/products/hero.jpg?w=480 480w, 
             /iar/products/hero.jpg?w=800 800w" 
     sizes="(max-width: 600px) 480px, 800px"
     alt="SEO Оптимизированное фото">
```

## ✨ Ключевые возможности

- **Максимальная портативность**: Работает на любом хостинге с PHP + GD. Не требует ImageMagick в системе.
- **Авто-формат**: Автоматически отдает WebP или AVIF, если браузер их поддерживает (через заголовок `Accept`), путь при этом остается прежним.
- **Умное кеширование**: Уникальные ключи на основе хеша пути и времени изменения оригинала.
- **Безопасность**: Защита от Path Traversal, DoS-атак (лимиты размеров) и XSS в SVG-файлах.
- **Гибкость**: Обрезка краев (в px или %), режимы заполнения (`cover`) с выбором точки фокуса и поддержка сжатия без потерь.

## ⚙️ Системные требования

- PHP 7.4 или выше (8.1+ рекомендуется для поддержки AVIF)
- Расширение PHP GD
- Apache с включенным `mod_rewrite` (для красивых ссылок)
- `php-imagick` (опционально, только для конвертации SVG в растр)
